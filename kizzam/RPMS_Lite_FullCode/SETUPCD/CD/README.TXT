To Install the Racing Pigeon Management system run 'SETUP.EXE'.

To Install Videos run 'SETUP.EXE' in the 'InstallVideos' folder.

To run videos from CD directly run from 'VIDEOS' folder on the CDROM.

To install the upgrade (ONLY FOR pre v3.2z users). Run setup.exe
from the \UPG32z folder. Once installed run the upgrade.exe to
upgrade you existing pre v3.2z database & files.
