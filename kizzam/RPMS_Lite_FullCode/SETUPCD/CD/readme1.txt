Setup.exe      installs a full version of the software
\installvideos installs the training video application
\UPG32z\setup  if exists on the CDROM, installs the Upgrade.exe program

If you are upgrading from a version pre V3.2z then you
MUST install the upgrade application then run
the upgrade.exe file. The upgrade makes changes to your
database, reports etc to ensure it is compatible with
new new version. If you are on RPMS V3.2z then DO NOT install the upgrade.
